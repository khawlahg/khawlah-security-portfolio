Key Insights

Genetic Algorithm Performance

  • GA achieved the highest fitness score through evolutionary optimization
  • Balanced detection rate and false positive rate effectively
  • Explored the search space efficiently using population-based approach
  • Demonstrated convergence over multiple generations

Search Algorithm Comparison

  • Greedy search found local optima quickly but may miss global best
  • A* search used heuristic guidance for efficient exploration
  • Both search methods outperformed fixed thresholds
  • GA's stochastic nature helped escape local optima

Fitness Function

fitness = detection_rate − false_positive_rate

This fitness function maximizes detection while penalizing noisy rules that generate false alarms.
